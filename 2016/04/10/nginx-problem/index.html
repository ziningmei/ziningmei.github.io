mac 安装nginx

  1. brew install nginx

  2. 制作软连接
    sudo ln /usr/local/Cellar/nginx/1.8.1/bin/nginx /usr/local/bin/nginx

  3. 配置文件地址/usr/local/nginx/conf/nginx.conf
